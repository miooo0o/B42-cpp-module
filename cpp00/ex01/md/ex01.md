	FirstName,

이제 `typedef`를 사용하지 않는다.

2.
```cpp
class PhoneBook {
public:
private:
};
```
`private`은 `public`을 통해서만 접근이 가능하다! 


3.
```cpp
public:
	Contact();	// Constructor
	~Contact();	// Destructor
```
**Constructor**:  `class`가 생성될 때 호출되는 functions.
**Destructor**:   `class`가 파괴될 때 호출되는 functions.
